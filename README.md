Проектная деятельность - веб-приложение Mesto.

Приложение Mesto. Это интерактивная веб-страница, куда можно добавлять фотографии. Также фотографии можно оценивать лайком и удалять.
При тестировании необходимо: 
  1. Провести регрессионное тестирование.
  2. Провести ретест.
Тестовый стенд - https://code.s3.yandex.net/qa/files/mesto/index.html
